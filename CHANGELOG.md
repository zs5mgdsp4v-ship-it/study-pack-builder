# Changelog

## 0.1.1 - 2026-06-16

- Add CSV validation for missing words, missing definitions, and duplicate vocabulary entries.
- Add answer-key scaffolding to quiz draft output.
- Add GitHub Actions CI for Python 3.10, 3.11, and 3.12.
- Add contribution, security, and issue-template documentation.

## 0.1.0 - 2026-06-16

- Add initial CLI package.
- Support vocabulary CSV to Markdown study packs.
- Support quiz draft generation from text.
- Support optional PDF output and PDF lightening helper.
- Add tests, examples, MIT license, and project README.
