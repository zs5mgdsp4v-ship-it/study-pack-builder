# Printable Vocabulary PDF Tables

`study-pack-builder vocab-table-pdf` creates compact landscape PDF tables for English vocabulary lists. The output is designed for classroom printing, quick review, and teacher handouts.

## Example

```bash
study-pack-builder vocab-table-pdf examples/english_academy_vocab.csv outputs/academy-vocab-table.pdf --title "English Academy Vocabulary Table"
```

## Table Columns

- Number
- Word
- Definition
- Synonyms
- Example

## Classroom Use

The PDF format is useful when teachers need a stable print layout for repeated review sessions. For editable material, generate Markdown with `vocab` or `review-sheet` first, then create PDF tables once the vocabulary content is reviewed.
