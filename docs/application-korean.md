# Korean Application Copy

These Korean drafts are written for maintainers applying with an early-stage but actively maintained open-source project.

## Version A: Balanced

### 이 리포지터리가 적합한 이유

study-pack-builder는 제가 주 책임자로 관리하는 초기 오픈소스 Python CLI입니다. 한국 영어학원과 시험 대비 교육 현장에서는 단어장, 암기장, 퀴즈, 인쇄용 복습지를 반복적으로 제작합니다. 이 프로젝트는 CSV, OCR, 텍스트 자료를 검증 가능한 학습팩과 퀴즈 초안으로 변환해 그 반복 작업을 줄입니다. MIT 라이선스, CI, 테스트, 릴리스, 보안 문서, 공개 로드맵 이슈를 갖추고 유지관리 중입니다.

### Codex Security가 필요한 이유

이 프로젝트는 사용자가 제공한 CSV, OCR 텍스트, PDF 파일을 처리하므로 안전한 파일 처리, 잘못된 입력값 검증, 의존성 위험 관리가 중요합니다. Codex Security를 통해 입력 검증, PDF 처리 코드, 의존성 업데이트, 릴리스 변경사항을 점검해 교사와 학생이 사용할 수 있는 안전한 로컬 CLI로 유지하고 싶습니다.

### API 크레딧 사용 계획

API 크레딧은 한국 영어학원 환경에서 필요한 OCR 단어장 정리, 단어 뜻과 예문 정제, 객관식/OX 퀴즈 생성, 정답 해설 작성, 수준별 복습지 변환 기능을 개선하는 데 사용할 계획입니다. Codex는 테스트 확장, CLI 구조 개선, 기여 리뷰, 보안 점검, 릴리스 자동화에 활용하겠습니다.

### 추가로 하고 싶은 말

이 프로젝트는 실제 학습자료 자동화 과정에서 시작했고, 지금은 공개된 로컬 우선 CLI로 정리했습니다. 아직 초기 단계이지만 릴리스, CI, 이슈 로드맵을 통해 계속 유지관리하고 있으며, 교사와 학생이 신뢰할 수 있는 학습팩 생성 도구로 발전시키겠습니다.

## Version B: Stronger Maintainer Focus

### 이 리포지터리가 적합한 이유

study-pack-builder는 한국 영어학원과 시험 대비 수업에서 반복되는 단어장, 퀴즈, 복습지 제작을 자동화하기 위한 오픈소스 CLI입니다. 아직 초기 프로젝트지만, 제가 주 책임자로 CI, 테스트, 보안 문서, 릴리스, 공개 로드맵 이슈를 관리하고 있습니다. Codex 지원을 통해 OCR 정리, 정답표 생성, 수준별 복습지, 릴리스 자동화를 빠르게 개선해 실제 교육 워크플로에 필요한 도구로 키우고자 합니다.

### Codex Security가 필요한 이유

학습자료에는 수업별 메모, OCR 결과, PDF 파일 등 다양한 사용자 입력이 포함될 수 있습니다. 로컬 CLI라도 잘못된 파일 처리, 깨진 CSV, PDF 처리 의존성, 향후 AI 기능의 데이터 처리 위험을 점검해야 합니다. Codex Security로 입력 검증, 의존성, 릴리스 변경사항을 지속적으로 검토하고 싶습니다.

### API 크레딧 사용 계획

API 크레딧은 영어 단어 OCR 정리, 뜻과 예문 정제, 객관식/OX 문제 생성, 정답 해설, 수준별 복습지 변환 기능에 사용할 계획입니다. 특히 한국 학원·시험 대비 환경에서 강사가 반복적으로 만드는 자료를 빠르게 초안화하고, Codex로 테스트와 리뷰를 확장해 안정적으로 유지관리하겠습니다.

### 추가로 하고 싶은 말

스타 수가 많은 성숙한 프로젝트는 아니지만, 실제 교육 현장의 반복 문제를 해결하는 유지관리 가능한 오픈소스 도구로 만들고 있습니다. 공개 릴리스와 이슈 로드맵을 기반으로 꾸준히 개선하고, 영어교육 자료 제작 워크플로를 더 안전하고 재사용 가능하게 만들겠습니다.
